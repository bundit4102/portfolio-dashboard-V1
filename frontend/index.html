ตารางกรอกข้อมูล Idea
# Idea ID ชื่อ Idea / Project Proposal ผู้เสนอ Category วันที่เสนอ งบประมาณ (฿) รายละเอียด / Business Case Impact (1-10)
🔵 Impact = ผลกระทบ / ประโยชน์ที่ได้ หมายถึง "ถ้าทำแล้วจะดีขึ้นแค่ไหน"
คะแนนสูง = ส่งผลดีมาก เช่น ลดต้นทุนเยอะ เพิ่มกำไร เพิ่ม productivity ชัดเจน
คะแนนต่ำ = ผลลัพธ์น้อย หรือแทบไม่เปลี่ยนอะไร
📌 ตัวอย่าง
ลดต้นทุนได้ 20% → Impact = 9–10
แค่ปรับเล็กน้อย → Impact = 3–4
 		Feasibility (1-10)
🟢 Feasibility = ความเป็นไปได้ / ทำได้จริงแค่ไหน หมายถึง "ทำได้ง่ายหรือยาก"
คะแนนสูง = ทำได้ง่าย ใช้ทรัพยากรน้อย ความเสี่ยงต่ำ
คะแนนต่ำ = ทำยาก ใช้เงินเยอะ ใช้เวลานาน หรือมีข้อจำกัดเยอะ
📌 ตัวอย่าง
ใช้ของเดิม ปรับ process เล็กน้อย → Feasibility = 8–10
ต้องลงทุนเครื่องจักรใหม่ → Feasibility = 3–5
 		Strategic (1-10)
🟠 Strategic = ความสอดคล้องกับกลยุทธ์องค์กร หมายถึง "ตรงกับเป้าหมายบริษัทไหม"
คะแนนสูง = ตรงกับ KPI / direction ขององค์กร เช่น Safety, Cost, Growth
คะแนนต่ำ = ดีแต่ไม่ใช่สิ่งที่องค์กรโฟกัสตอนนี้
📌 ตัวอย่าง
บริษัทเน้นลด cost แล้วโปรเจกต์ช่วยลด cost → Strategic = 9–10
เป็นไอเดียดีแต่ไม่ใช่ priority → Strategic = 4–5
 		Score สถานะ หมายเหตุ
